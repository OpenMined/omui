# Primitives

Primitives are components within OpenMined UI that are intended to mask existing HTML functionality, like composing a `<div>` (the `<Box>` component), implementing Flexbox (the `<Flex>` component), or displaying an image (the `<Image>` component) to name a few. In the following series of documents, we will describe these primitives and how they should be used in the context of developing a web application UI with OMUI.

It's suggested that in order to understand the concept of primitives, that you first understand how styled props in Styled System (and Chakra) work. We suggest you read the following pieces of Chakra documentation before continuing on:

- [Chakra Design Principles](https://chakra-ui.com/principles)
- [Styled Props](https://chakra-ui.com/style-props)
- [Responsive Styles](https://chakra-ui.com/responsive-styles)

The current list of primitives for OMUI are based, more or less, on [Rebass](https://rebassjs.org/) (another [Styled System](https://styled-system.com/) project). Since each primitive is also present in Chakra, we will also link to their documentation below. Briefly, they are as follows:
